Desafio Kabum:

Pre-requisitos:
Navegador Firefox e selenium-webdriver

Dependencias utilizadas: Cucumber, Ruby, Selenium, Capybara e Rspec

Para o desenvolvimento do projeto utilizei a linguagem Ruby junto com Capybara e Rspec, utilizando o BDD Cucumber. Foi proposto o login no site Kabum e validação de usuário logado com sucesso.

Para a execução do projeto, basta executar os seguintes comandos:

Abrir terminal
Navegar até a pasta do projeto "Desafio Kabum"
Rodar o comando gem install bunler
Rodar o comando bundle install
Rodar o comando cucumber features/login.features

Espero que gostem!
